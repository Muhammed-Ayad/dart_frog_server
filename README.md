# Dart Frog 

## Installation

```bash
   dart pub global activate dart_frog_cli
   dart_frog create my_project
```
    
## Run Server 

```bash
   dart_frog dev
```

## Create Routes 


### Absolute Path
- path like "/" or "/hello" or "/hello/world"
- path /hello/world  will craete folder named ``hello`` and file named ``word.dart``

```bash
   dart_frog new route <path>
```
